# 丁主任 - 智能疫病监测预警机器人平台

## 项目概述
基于人兽共患病监测预警的 AI Agent 平台

## 核心功能
- 疫病监测预警机器人硬件联动
- 行业资讯自动爬取与分析
- 知识星球（行业今日头条）
- 投行研报自动生成

## 目标行业
- 疫病疾控行业（核心）
- 农业行业（扩展）

## 文件夹结构
- 需求文档/ - 项目方案与需求
- 参考资料/ - 技术资料与文献
- 行业数据/ - 数据库与数据资源
- 原型设计/ - UI/UX 设计稿

## 时间线
- 2022: 机器人模型1.0 + 国家发明专利
- 2024: 第二代原型机 + 国家发明专利
- 2025: 第三代原型机 + 美国发明专利 + 5项软件著作权
